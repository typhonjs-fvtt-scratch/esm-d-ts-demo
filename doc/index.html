esm-d-ts-demo

This is a demo package showing configuration of esm-d-ts to generate types from ESM source code.

This trial shows that Typescript currently requires .mjs files when using import types to include the extension .mjs. For example see here

You can see the output of type generation as follows between .js and .mjs files here:

  • src/js/index.d.ts
  • src/mjs/index.d.ts

esm-d-ts is configured as a NPM script here in package.json
